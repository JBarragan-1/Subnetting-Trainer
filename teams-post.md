# 🎮 Subnetting Trainer is live — learn (or refresh) IPv4 subnetting in the browser

Hi team,

Sharing a new **interactive Subnetting Trainer** — a single HTML file you can open in any browser, no install, no signup. Built to make subnetting practice feel less like flashcards and more like a game.

## What's inside

- **6 missions** — Binary Bootcamp → Class Sorting → Mask Match → Host Count → Network ID → VLSM
- **Each mission has 3 stages** — read the study guide, practice with a hands-on lesson, then take the scored challenge
- **🧪 Sandbox** — type any IP/prefix to see network ID, broadcast, hosts, mask, and binary breakdown
- **🗺️ Topology Builder** — drop routers and switches on a canvas, link them up, and watch VLSM allocate subnets live
- **⚡ Challenge Mode** — three difficulty tiers (Veteran, Expert, Nightmare) for when you want to compete with yourself
- **🐉 Final Boss** — locked until all 6 missions clear; tests every question type at once
- **🏅 15 achievements** — streaks, perfect runs, speed records, and the elusive "No Mistakes" boss clear

## Why it's different

Question pools randomize on every attempt, so replays don't turn into rote memorization. Lessons are deliberately un-timed and forgiving — the timer pressure lives in the missions, where it belongs. The topology builder shows you wasted addresses in real time, which is the actual metric that matters when you're planning real networks.

## How to use it

1. Download `subnetting-trainer.html`
2. Double-click to open
3. The 30-second tour starts automatically

## Suggested goals

- **Newer to networking:** clear all 6 missions and beat the boss
- **Refreshing:** skip to Sandbox + Topology, then run Veteran tier
- **Going for it:** clear Nightmare tier and grab the No Mistakes boss badge — that one signals real fluency

Roughly 60–90 minutes start to finish. Replays are fresh because of the question pools, so it doubles as ongoing practice.
